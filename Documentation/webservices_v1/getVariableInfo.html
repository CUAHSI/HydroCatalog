Description

getVariableInfo: Given a variable code, this method returns the variable's name. Pass in the variable in this format: 'NetworkName:Variable'. Sending no variable, or an empty variable will return a list of all variables.

Example Endpoint WSDL

Little Bear River: http://his02.usu.edu/littlebearriver/cuahsi_1_0.asmx?WSDL

Input


Parameters Variable TYPE EXAMPLE
NetworkName:SiteCode

String

"LBR:USU-LBR-Mendon"
Authentication String

String

""

Output


PARAMETERS VARIABLE TYPE PATH DESCRIPTION EXAMPLE
GetVariableInfoResponse

String

GetVariableInfoResponse/...

Returns a variable list from selected site as an XML string

See SOAP RESPONSE
