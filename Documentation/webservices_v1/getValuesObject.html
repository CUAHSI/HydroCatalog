Description

getValuesObject: Given a site number, a variable, a start date, and an end date, this method returns a time series. Pass in the sitevode and variable in this format: 'NetworkName:SiteCode' and 'NetworkName:SiteCode' and 'NetworkName:Variable'. Sending a null BeginDate and EndDate will return all values for the given parameters. This may not work with all services.

Example Endpoint WSDL

Little Bear River: http://his02.usu.edu/littlebearriver/cuahsi_1_0.asmx?WSDL
