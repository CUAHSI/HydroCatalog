  • Mailing List

Description

getSiteInfo: Given a site number, this method returns the site's metadata. Send the site code in this fomrat: 'NetworkName:SiteCode'.

Example Endpoint WSDL

Little Bear River: http://his02.usu.edu/littlebearriver/cuahsi_1_0.asmx?WSDL

Input


Parameters Variable TYPE EXAMPLE
NetworkName:SiteCode

String

"LBR:USU-LBR-Mendon"
Authentication String

String

""

Output


PARAMETERS VARIABLE TYPE PATH DESCRIPTION EXAMPLE
siteInfo

Object

sitesResponse/site/...

Contains information about the site (See SUB-PARAMETERS).

See SOAP RESPONSE
seriesCatalog

Object

sitesResponse/site/...

Contains information about series contained in sites' data.

See SOAP RESPONSE
SUB-PARAMETERS VARIABLE TYPE PATH DESCRIPTION EXAMPLE
siteName

String

./site/siteInfo/...

Name of the site

See SOAP RESPONSE
siteCode

String

./site/siteInfo/...

Code of the site

See SOAP RESPONSE
geoLocation

Object

./site/siteInfo/...

Latitude and longitude of the site

See SOAP RESPONSE
verticalDatum

String

./site/siteInfo/...

Elevation measurement reference

See SOAP RESPONSE
note

String

./site/siteInfo/...

Information such as county, state and site description

See SOAP RESPONSE

SOAP REQUEST 


<soapenv:Envelope xmlns:soapenv="http://schemas.xmlsoap.org/soap/envelope/" xmlns:ws="http://www.cuahsi.org/his/1.0/ws/">
   <soapenv:Header/>
   <soapenv:Body>
      <ws:GetSiteInfoObject>
         <!--Optional:-->
         <ws:site>LBR:USU-LBR-Mendon</ws:site>
         <!--Optional:-->
         <ws:authToken>""</ws:authToken>
      </ws:GetSiteInfoObject>
   </soapenv:Body>
</soapenv:Envelope>
