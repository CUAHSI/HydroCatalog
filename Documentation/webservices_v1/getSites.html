Description

getSites: Given an array of site numbers, this method returns the site metadata for each one. Send the array of site codes in this format: 'NetworkName:SiteCode'. Sending an empty array will return all sites, up to a limit of 50,000.

Example Endpoint WSDL

Little Bear River: http://his02.usu.edu/littlebearriver/cuahsi_1_0.asmx?WSDL

Input


Parameters Variable TYPE EXAMPLE
NetworkName:SiteCode

String

""
Authentication String

String

""

Output


PARAMETERS VARIABLE TYPE PATH DESCRIPTION EXAMPLE
sitesResponse

Object

...

Returns a list of sites as an XML string

See SOAP RESPONSE
SUB-PARAMETERS VARIABLE TYPE PATH DESCRIPTION EXAMPLE
site

Object

./sitesResponse/...

Returns all sites as objects for the above input parameters.

See SOAP RESPONSE

SOAP REQUEST 


<soapenv:Envelope xmlns:soapenv="http://schemas.xmlsoap.org/soap/envelope/" xmlns:ws="http://www.cuahsi.org/his/1.0/ws/">
   <soapenv:Header/>
   <soapenv:Body>
      <ws:GetSites>
         <!--Optional:-->
         <ws:site>
            <!--Zero or more repetitions:-->
            <ws:string></ws:string>
         </ws:site>
         <!--Optional:-->
         <ws:authToken>""</ws:authToken>
      </ws:GetSites>
   </soapenv:Body>
</soapenv:Envelope>
